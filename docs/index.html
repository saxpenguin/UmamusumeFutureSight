隱私權政策

本應用程式（以下簡稱「本 App」）尊重並保護所有使用者的文字隱私權。為了幫助您了解本 App 如何蒐集、應用及保護您所提供的個人資訊，請詳細閱讀以下隱私權政策。

1. 資料蒐集之目的

本 App 為《賽馬娘》輔助工具，僅用於提供遊戲資訊查詢。我們不會主動要求、儲存或傳輸您的任何個人識別資料（如姓名、電話、電子郵件）。

2. 第三方服務

本 App 可能使用以下第三方服務，這些服務可能會根據其隱私權政策蒐集非識別性資訊：

  • Firebase Remote Config：用於更新遊戲參數設定。
  • Google Play Services：用於 App 運行與必要之系統更新。

3. 資料安全性

我們採取合理的步驟來保護資料免於遺失、誤用或未經授權的存取。由於本 App 不蒐集個人隱私資料，因此不會有個人資料洩露之風險。

4. 政策變更

本隱私權政策可能會不時更新。建議您定期閱覽本頁面以了解最新變動。

5. 聯絡我們

若對本隱私權政策有任何疑問，請透過 GitHub 專案頁面或電子郵件聯絡開發者。
